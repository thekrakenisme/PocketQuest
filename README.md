# PocketQuest

PocketQuest currently contains a combat vertical slice focused on reproducing the pacing of classic EverQuest-style fights in a mostly text-first format.

## Run

Open [index.html](C:\Users\thekr\Desktop\Projects\Personal Projects\PocketQuest\index.html) in a browser, or serve the folder with any static file server.

## Included

- [index.html](C:\Users\thekr\Desktop\Projects\Personal Projects\PocketQuest\index.html): self-contained simulator entry point
- [src/app.js](C:\Users\thekr\Desktop\Projects\Personal Projects\PocketQuest\src\app.js): combat engine and UI rendering
- [src/styles.css](C:\Users\thekr\Desktop\Projects\Personal Projects\PocketQuest\src\styles.css): mobile-first presentation
- [Docs/combat-research-and-plan.md](C:\Users\thekr\Desktop\Projects\Personal Projects\PocketQuest\Docs\combat-research-and-plan.md): transferred research and design notes
